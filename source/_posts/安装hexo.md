# 安装
## 安装nodejs
前往nodejs官网 https://nodejs.org/zh-cn 下载安装包
### 查看nodejs和npm版本
```bash
node -V
npm -V
```
- 查看nodejs版本
```bash
node  -V
```
- 查看npm版本

```bash
npm  -V
```

## 安装hexo工具

```bash
npm install hexo-cli -g
```
## 创建项目

```bash
hexo insatll projext_name
```
## 安装依赖
```bash
cd project_name
npm i
```
# 创建页面
```bash
hexo new [layout] <title>
```
# 运行
```bash
npm run server
```
